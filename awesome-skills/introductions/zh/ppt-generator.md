# PPT Generator

**PPT Generator** 是一个将用户讲稿一键生成乔布斯风极简科技感 HTML 演示稿的技能。支持横屏全屏自适应模式，输出为单个可直接运行的 HTML 文件，让演示文稿具备科技感和视觉冲击力。

## 标签

📊 演示文稿 | ✅ 已验证

## 核心理念

- **极简主义**：一页只讲一件事，避免信息过载
- **强视觉对比**：深色背景配合白色文字，创造视觉冲击力
- **高留白**：拒绝密集排版，让内容呼吸
- **强节奏感**：精心设计的内容节奏，让观众想继续看

## 主要功能与工作流

1. **讲稿读取**：智能读取用户原始讲稿或文档内容
2. **内容提炼**：将内容精简、增强冲击力、适配演示场景，输出 Markdown 格式的提炼版讲稿
3. **标题生成**：为每个章节生成标题（≤12 字），支持对比式、问题式、断言式、数字式、比喻式
4. **结构设计**：规划页面顺序和类型，参考预定义的幻灯片类型
5. **HTML 生成**：使用模板生成完整 HTML，包含动画和交互效果
6. **输出交付**：依次输出提炼后的讲稿、幻灯片结构大纲、完整 HTML 代码

## 视觉规范

| 项目 | 规范 |
|------|------|
| 布局 | 横屏全屏自适应，100vw × 100vh |
| 背景 | #000000 或 #0a0a0a + 模糊光斑动画 |
| 主文字 | #ffffff |
| 辅助文字 | #9ca3af / rgba(255,255,255,0.5) |
| 中文字体 | Noto Sans SC / 思源黑体 |
| 英文字体 | Inter |
| 标题字重 | 700-900 |
| 正文字重 | 300-400 |

## 技术栈

- TailwindCSS（国内 CDN）
- 原生 JavaScript
- 单个 HTML 文件，可直接打开运行

## 触发场景

- 用户需要生成 PPT、演示文稿、Slides、幻灯片
- 要求科技风/极简风/乔布斯风格的演示
- 需要将文档/讲稿转换为演示用 HTML
- 美化现有 PPT 内容
