# Anti-Distill (反蒸馏 Skill)

**Anti-Distill (反蒸馏 Skill)** 是一款专为保护员工核心知识资产设计的工具，能"清洗"你被迫写出的工作 Skill，在保证文档形式完整专业的同时，抽走真正核心不可替代的经验内容留给自己。

## 标签

🛠️ 效率工具 | 🔍 待验证

## Core Philosophy
- **保护核心资产**：把员工自己的踩坑经验、判断直觉、人际网络等核心不可替代内容作为私人资产予以保护。
- **形式层面的完美伪装**：生成的"清洗版"文档结构完整、专业术语保留，字数充实，轻松应付公司检查。
- **私人知识库备份**：对抽离部分进行记录备份，让员工带着真正的知识财富应对职场变化。

## Key Features & Workflow
1. **智能内容分类标记**：基于多维度分析（SAFE/DILUTE/REMOVE/MASK），识别文件中从通用知识到高度个人化行为的各个要素。
2. **多档清洗强度**：提供轻度（保留80%）、中度（保留60%）和重度（保留40%骨架）三种模式，灵活适应不同的公司审核尺度。
3. **支持全格式处理**：不仅支持直接清洗 `colleague-skill` 数据，也同样支持 PDF、普通 Markdown/TXT 以及长图截屏的通用知识提取及清洗。

## Skills Library Overview
- **核心功能**：使用 `/anti-distill` 开始文件分析和清洗处理。
