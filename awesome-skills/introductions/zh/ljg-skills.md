# LJG Skills

**LJG Skills** 是由李继刚打造的个人 AI 技能集，包含 20 个原子技能和 2 个工作流，覆盖内容铸造、概念解剖、论文研读、写作引擎、演讲铸造、伴读引导等领域——把 AI 变成你的认知手术刀。

## 标签

🧘 人生与哲学 | ✅ 已验证

## 核心理念

LJG Skills 的设计哲学围绕"知识炼金术"展开：
- **内容铸造成形**：文字进去，PNG 出来。模具决定形状——长图、信息图、漫画、白板、大字碑刻，七种模具任选。
- **手术刀式写作**：对准一个观点下刀，一层层剥开，挖到底。不罗列要点，而是层层推进、持续深入。
- **概念八维解剖**：从历史、辩证、现象、语言、形式、存在、美感、元反思八个方向切开一个概念，压成一句顿悟。
- **反 AI 痕迹**：禁 Inter 字体、禁纯黑、禁三等分卡片、禁居中 Hero、禁 AI 文案腔——所有输出追求人的品味。

## 主要功能与工作流

1. **内容铸卡 (ljg-card)**：将任意内容转为 PNG 视觉卡片，支持 7 种模具——长图、信息图、多卡、视觉笔记、漫画、白板、大字碑刻。
2. **概念解剖 (ljg-learn)**：从八个方向解剖一个概念，从定锚到八刀、内观、压缩，最终输出一句顿悟。
3. **论文研读 (ljg-paper)**：为非学术人士提取论文核心想法，从亲历困境到博导审稿，重理解不重批判。
4. **论文溯源 (ljg-paper-river)**：倒读法递归挖前序论文（最多 5 层）+ 最新进展，从源头讲述问题演化史。
5. **写作引擎 (ljg-writes)**：像手术刀剖开一个观点，层层剥到底，1000-1500 字的批判性深度文章。
6. **白话引擎 (ljg-plain)**：把任何内容改写到聪明的十二岁小孩也能懂。
7. **信息提问机 (ljg-qa)**：把核心观点抽成 Q-A 链，Q 切要害，A 四段展开（结论 / 形式化 / 步骤 / 边界）。
8. **降秩引擎 (ljg-rank)**：给一个领域，找出背后不可再少的独立生成器。
9. **追本之箭 (ljg-think)**：给一个观点或现象，纵向深钻到不可再分的本质。
10. **演讲铸造器 (ljg-present)**：高桥流（一页一词）或标语流（色块宣言），输出单文件 HTML 演示文稿。
11. **伴读 (ljg-read)**：陪你读任何文本，英文三层翻译（信达雅）+ 结构标注 + 深度提问 + 跨领域旁逸。
12. **单词精通 (ljg-word)**：深度拆解一个英语单词的核心语义和顿悟时刻。
13. **投资分析 (ljg-invest)**：核心判断项目是否是一台"秩序创造机器"。
14. **关系分析 (ljg-relationship)**：五层结构诊断 + 精神分析，帮用户"看见"关系真实结构。
15. **圆桌讨论 (ljg-roundtable)**：求真导向的结构化多人辩证对话，每轮生成 ASCII 思考框架图。
16. **旅行研究 (ljg-travel)**：输入城市名，生成深度文化研究文档 + 便携卡片。
17. **技能地图 (ljg-skill-map)**：扫描所有已安装技能，渲染可视化总览。
18. **推送引擎 (ljg-push)**：一键同步本地技能到 GitHub 仓库。

## 技能库概览

- **内容铸造**：ljg-card（视觉卡片）、ljg-present（演讲铸造）
- **阅读研究**：ljg-paper（论文研读）、ljg-paper-river（论文溯源）、ljg-read（伴读）
- **写作思考**：ljg-writes（写作引擎）、ljg-plain（白话引擎）、ljg-qa（信息提问机）
- **概念探索**：ljg-learn（概念解剖）、ljg-think（追本之箭）、ljg-rank（降秩引擎）、ljg-word（单词精通）
- **专业分析**：ljg-invest（投资分析）、ljg-relationship（关系分析）
- **对话互动**：ljg-roundtable（圆桌讨论）
- **工具输出**：ljg-travel（旅行研究）、ljg-skill-map（技能地图）、ljg-push（推送引擎）
- **工作流**：ljg-paper-flow（读论文 + 做漫画卡片）、ljg-word-flow（单词分析 + 信息图卡片）
