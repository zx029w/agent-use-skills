# MiniMax Skills

**MiniMax Skills** 是一套面向 AI 编程工具的综合技能库，提供结构化的、生产级质量的前端、全栈、移动端（Android、iOS、Flutter、React Native）、着色器开发以及文档处理（PDF、PPTX、XLSX、DOCX）指导。

## 标签

🗂️ 文档与办公 | ✅ 已验证

## 核心理念

MiniMax Skills 的设计哲学：
- **结构化指导**：提供生产级质量的开发工作流和最佳实践
- **全栈覆盖**：从移动端到 Web，从后端到视觉特效的全方位支持
- **多模态能力**：集成 MiniMax API 的图片、视频、音频、音乐生成能力
- **文档处理**：专业级的 PDF、PPT、Excel、Word 文档创建和编辑

## 技能库概览

### 📱 移动开发
- **android-native-dev**：基于 Material Design 3 的 Android 原生应用开发（Kotlin/Jetpack Compose）
- **ios-application-dev**：iOS 应用开发指南（UIKit、SnapKit、SwiftUI）
- **flutter-dev**：Flutter 跨平台开发（Riverpod/Bloc 状态管理）
- **react-native-dev**：React Native 和 Expo 开发指南

### 🌐 Web 开发
- **frontend-dev**：全栈前端开发，融合高级 UI 设计、电影级动画、AI 媒体生成（React/Next.js、Tailwind CSS）
- **fullstack-dev**：全栈后端架构与前后端集成（REST API、认证、实时功能、数据库）

### 🎨 视觉与创意
- **shader-dev**：全面的 GLSL 着色器技术，用于创建惊艳的视觉效果（兼容 ShaderToy）
- **gif-sticker-maker**：将照片转换为 Funko Pop 风格的动画 GIF 贴纸
- **vision-analysis**：使用视觉 AI 模型分析、描述和提取图像信息

### 📄 文档处理
- **minimax-pdf**：基于 token 化设计系统生成、填写和重排 PDF 文档
- **pptx-generator**：生成、编辑和读取 PowerPoint 演示文稿
- **minimax-xlsx**：Excel/电子表格文件的创建、读取、分析和编辑
- **minimax-docx**：专业 DOCX 文档的创建、编辑与排版

### 🎵 多模态
- **minimax-multimodal-toolkit**：通过 MiniMax API 生成语音、音乐、视频和图片内容

## 主要功能与工作流

1. **移动应用开发**：涵盖 Android、iOS、Flutter、React Native 的完整开发指南，包括 UI 设计、性能优化、无障碍支持
2. **全栈 Web 开发**：从前端动画到后端架构的端到端指导
3. **视觉特效创作**：GLSL 着色器编程，创建电影级视觉效果
4. **文档生成与处理**：专业级的 Office 文档创建和编辑能力
5. **多模态内容生成**：集成 MiniMax API 的 AI 媒体生成能力
