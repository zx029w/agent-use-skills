# Baoyu Skills

**Baoyu Skills** 是宝玉分享的一套为 Claude Code 等 AI 编程助手量身打造的效率提升技能集，涵盖内容创作、图像生成、社交媒体发布、文档处理等多个维度的实用工具。

## 标签

🎨 设计与创意 | ✅ 已验证

## 核心理念
Baoyu Skills 的设计哲学体现在以下几点：
- **内容即生产力**：将文章、图片、视频等内容的生产和发布全流程 AI 化。
- **多平台覆盖**：支持小红书、微信公众号、微博、X（Twitter）等主流社交平台的内容发布。
- **灵活可扩展**：通过 EXTEND.md 机制支持项目级和用户级的自定义配置，满足不同团队和个人的需求。
- **多后端支持**：图像生成等核心功能支持 OpenAI、Google、DashScope、MiniMax、Replicate 等多种 AI 服务商。

## 主要功能与工作流

1. **内容创作与视觉化（Content & Visual）**：
   - `baoyu-xhs-images`：小红书图文信息图生成器，支持 9 种视觉风格和 6 种布局。
   - `baoyu-infographic`：专业信息图生成工具，拥有 20 种布局和 17 种视觉风格。
   - `baoyu-cover-image`：文章封面图生成工具，基于 5 维度（类型×配色×渲染×文字×情绪）系统。
   - `baoyu-slide-deck`：从文章内容生成专业幻灯片图集，自动合并为 .pptx 和 .pdf。
   - `baoyu-comic`：知识漫画创作工具，支持多种漫画艺术风格和情绪基调。
   - `baoyu-article-illustrator`：智能文章配图工具，自动分析文章结构并在合适位置插入图示。

2. **社交媒体发布（Social Publishing）**：
   - `baoyu-post-to-x`：发布内容到 X（Twitter），支持普通推文和 X 长文章格式。
   - `baoyu-post-to-wechat`：发布内容到微信公众号，支持图文和文章两种模式，可配置多账号。
   - `baoyu-post-to-weibo`：发布内容到微博，支持普通帖子和头条文章格式。

3. **AI 图像生成（AI Image Generation）**：
   - `baoyu-imagine`：多后端 AI 图像生成工具，支持 OpenAI、Azure、Google、DashScope、MiniMax、Jimeng、Seedream、Replicate 等，支持参考图、批量生成和自定义尺寸。
   - `baoyu-danger-gemini-web`：通过 Gemini Web 生成文本和图像。

4. **内容处理工具（Content Utilities）**：
   - `baoyu-youtube-transcript`：下载 YouTube 视频字幕，支持多语言、翻译、章节和说话人识别。
   - `baoyu-url-to-markdown`：通过 Chrome CDP 抓取任意页面并转换为 Markdown。
   - `baoyu-danger-x-to-markdown`：将 X (Twitter) 内容（推文和长文章）转为 Markdown 格式。
   - `baoyu-compress-image`：压缩图片，在保持质量的前提下减小文件体积。
   - `baoyu-format-markdown`：格式化 Markdown 文件，自动处理 frontmatter、标题、摘要等。
   - `baoyu-markdown-to-html`：将 Markdown 转换为带有微信兼容主题的 HTML。
   - `baoyu-translate`：多模式文章翻译工具，支持快速、普通和精炼三种翻译模式。

## 技能库概览
- **内容创作**：小红书图文、信息图、封面图、幻灯片、漫画、文章配图。
- **社交发布**：X (Twitter)、微信公众号、微博 的内容发布自动化。
- **图像生成**：支持多个主流 AI 图像服务商的统一调用接口。
- **内容工具**：YouTube 字幕、URL 抓取、格式转换、图片压缩、文章翻译。
