# Slidev

**Slidev** 是一款基于 Vite、Vue 和 Markdown 构建的面向开发者的幻灯片制作和演示工具。

## 标签

🎨 设计与创意 | 🔍 待验证

## 核心理念
- **开发者体验优先**: 使用 Markdown 和 Vue 语法编写 PPT，拥抱前端生态。
- **丰富的技术演示**: 原生支持代码高亮、交互式代码执行和数学公式排版，专门服务于技术研发者。
- **现代化互动**: 支持高级动画过渡、多布局支持和实时录制演示。
- **全平台支持**: 一次编写，可部署为单页应用 (SPA)，或导出为 PDF / PPTX。

## 主要功能与工作流
1. **纯文本驱动**: 使用清晰分离的 Markdown 文件快速起草和构建幻灯片。
2. **丰富的组件集成**: 支持集成任意 Vue 组件、Twoslash 类型提示和动画（例如 `v-click`，动态转场配合）。
3. **技术特性**: 直接使用内置的 Monaco 编辑器、Mermaid/PlantUML 流程图、LaTeX 公式。
4. **演示管理**: 利用强大的 Presenter 模式，随时控制幻灯片进程，查阅演讲者笔记或直接开启内置录制功能。

## 技能库概览
- **自动化起步**: 提供初始化项目、开发环境托管和构建导出的端到端能力集。
- **模板与排版**: 涵盖大量官方排版与布局结构引用 (`core-layouts`, `core-components` 等)。
