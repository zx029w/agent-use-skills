# Colleague Skill

**Colleague Skill** 是一个将同事"蒸馏"为 AI Skill 的工具，通过自动采集飞书/钉钉数据，生成包含工作能力和性格特征的数字化同事。

## 标签

🛠️ 效率工具 | 🔍 待验证

## Core Philosophy
- **全面还原**：不仅还原工作技能，更注重同事的独立性格、表达逻辑与决策模式。
- **动态进化**：支持追加文件、对话纠正版本管理，随接触不断丰富数字人的全貌。
- **多源融合**：兼顾企业聊天记录、内部文档以及个人手动补充的特点。

## Key Features & Workflow
1. **多模态数据采集**：支持飞书、钉钉自动拉取并深度解析，也可补充聊天截图与邮件。
2. **两层架构（Work & Persona）**：将技术资产（工作规范、专业知识）和情感连接（口头禅、处事态度）分离后叠加输出。
3. **对话式进化机制**：发现"他不会这样说"，可直接进行语义修正并实现局部更新，不破坏原有能力。

## Skills Library Overview
- **创建与管理**：使用 `/create-colleague` 创建。
- **更新与删除**：使用 `/update-colleague`、`/list-colleagues` 等命令维护同事数字生命仓库。
