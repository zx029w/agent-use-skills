# Anti-Distill

**Anti-Distill** is a tool designed to clean your forced-to-write employee AI Skills, ensuring the handed-in files look complete and professional while the core knowledge assets are kept strictly to yourself.

## Tags

🛠️ Productivity Tools | 🔍 Pending Verification

## Core Philosophy
- **Protect Core Assets**: Safeguards irreplaceable employee knowledge like pitfall experiences, intuition, and interpersonal context as private assets.
- **Perfect Structural Camouflage**: The "cleaned" version has complete document structure and proper terminology intact to pass corporate reviews flawlessly.
- **Private Knowledge Backup**: Records all extracted knowledge elements into a private backup, preserving true professional wealth for future careers.

## Key Features & Workflow
1. **Intelligent Content Classification**: Uses multi-dimensional analysis (SAFE/DILUTE/REMOVE/MASK) to identify elements ranging from generic knowledge to highly personalized behaviors.
2. **Multiple Cleaning Intensities**: Offers Light (80% retention), Medium (60%), and Heavy (40% generic skeleton) modes adapting to different corporate audit levels.
3. **Universal Format Support**: Processes `colleague-skill` data naturally, while seamlessly reading generic Markdown, TXT, PDFs, or even images.

## Skills Library Overview
- **Core Action**: Trigger with `/anti-distill` to initiate file analysis and intelligent dilution.
