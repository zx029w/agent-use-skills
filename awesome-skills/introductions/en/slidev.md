# Slidev

**Slidev** is a presentation slides maker tailored for developers, built on Vite, Vue, and Markdown.

## Tags

🎨 Design & Creativity | 🔍 Pending Verification

## Core Philosophy
- **Developer Experience First**: Write slide decks using familiar Markdown and Vue ecosystem.
- **Geared for Tech**: Natively treats syntax-highlighted code, runnable snippets, and equations as first-class citizens.
- **Modern & Interactive**: Empowers presentations with rich animations, comprehensive layouts, and dynamic elements.
- **Cross-environment**: Build once, serve as an SPA, or export immediately to PDF/PPTX formats.

## Key Features & Workflow
1. **Markdown Driven**: Draft comprehensive presentations quickly with standard structured markdown formatting.
2. **Ecosystem Integration**: Drop in Vue components directly, utilize Twoslash hints, and wield precise animation control via `v-click`.
3. **Technical Out-of-the-Box**: Embed live Monaco editors, define logic charts with Mermaid/PlantUML, or typeset pure math via LaTeX natively.
4. **Presenter Empowered**: Utilize a robust presenter mode covering speaker notes, live recording, and granular presentation progression.

## Skills Library Overview
- **Project Lifecycle**: Covers everything from bootstrap and development to exporting and hosting.
- **Structure Ecosystem**: Access to rich references including layouts formatting (`core-layouts`), animation structures (`core-animations`), and internal components API (`core-components`).
