# MiniMax Skills

**MiniMax Skills** is a comprehensive skill library for AI coding agents, providing structured, production-quality guidance across multiple development domains including frontend, fullstack backend, mobile development (Android, iOS, Flutter, React Native), shader programming, and document processing.

## Tags

💻 Dev & Testing | ✅ Verified

## Core Philosophy

MiniMax Skills is designed around:
- **Structured Guidance**: Production-quality development workflows and best practices
- **Full-Stack Coverage**: From mobile to Web, backend to visual effects
- **Multimodal Capabilities**: Integrated MiniMax API for image, video, audio, and music generation
- **Document Processing**: Professional-grade PDF, PPT, Excel, and Word document creation and editing

## Skills Library Overview

### 📱 Mobile Development
- **android-native-dev**: Android native development with Material Design 3 (Kotlin/Jetpack Compose)
- **ios-application-dev**: iOS application development guide (UIKit, SnapKit, SwiftUI)
- **flutter-dev**: Flutter cross-platform development (Riverpod/Bloc state management)
- **react-native-dev**: React Native and Expo development guide

### 🌐 Web Development
- **frontend-dev**: Full-stack frontend with premium UI design, cinematic animations, AI-generated media (React/Next.js, Tailwind CSS)
- **fullstack-dev**: Backend architecture and frontend-backend integration (REST API, auth, real-time, database)

### 🎨 Visual & Creative
- **shader-dev**: Comprehensive GLSL shader techniques for stunning visual effects (ShaderToy-compatible)
- **gif-sticker-maker**: Convert photos into Funko Pop style animated GIF stickers
- **vision-analysis**: Analyze, describe, and extract information from images using vision AI

### 📄 Document Processing
- **minimax-pdf**: Generate, fill, and reformat PDF documents with token-based design system
- **pptx-generator**: Generate, edit, and read PowerPoint presentations
- **minimax-xlsx**: Create, read, analyze, edit, or validate Excel/spreadsheet files
- **minimax-docx**: Professional DOCX document creation, editing, and formatting

### 🎵 Multimodal
- **minimax-multimodal-toolkit**: Generate voice, music, video, and image content via MiniMax APIs

## Key Features & Workflow

1. **Mobile App Development**: Complete development guides for Android, iOS, Flutter, React Native covering UI design, performance optimization, and accessibility
2. **Full-Stack Web Development**: End-to-end guidance from frontend animations to backend architecture
3. **Visual Effects Creation**: GLSL shader programming for cinematic visual effects
4. **Document Generation & Processing**: Professional Office document creation and editing capabilities
5. **Multimodal Content Generation**: AI-powered media generation via MiniMax API integration
