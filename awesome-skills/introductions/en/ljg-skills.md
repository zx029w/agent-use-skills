# LJG Skills

**LJG Skills** is a personal AI skill collection created by Li Jigang, featuring 20 atomic skills and 2 workflows covering content casting, concept dissection, paper reading, writing engine, presentation forging, guided reading, and more — turning AI into your cognitive scalpel.

## Core Philosophy

LJG Skills is designed around the idea of "knowledge alchemy":
- **Content Casting**: Text goes in, PNG comes out. The mold determines the shape — long card, infographic, comic, whiteboard, big-font stele, and more. Seven molds to choose from.
- **Scalpel Writing**: Cut into an idea, peel it layer by layer, dig to the bottom. Not bullet-point lists, but progressively deeper critical thinking.
- **Eight-Dimensional Concept Anatomy**: Dissect a concept from eight directions — history, dialectics, phenomenology, linguistics, formalization, existentialism, aesthetics, and meta-reflection — then compress into a single epiphany.
- **Anti-AI Aesthetics**: No Inter font, no pure black, no tripartite cards, no centered hero sections, no AI copywriting tone — all output pursues human taste.

## Key Features & Workflow

1. **Content Casting (ljg-card)**: Transform any content into PNG visual cards with 7 molds — long card, infographic, multi-card, sketchnote, comic, whiteboard, and big-font stele.
2. **Concept Anatomy (ljg-learn)**: Dissect a concept from eight dimensions — from anchoring to eight cuts, introspection, and compression, outputting a single epiphany.
3. **Paper Reading (ljg-paper)**: Extract core ideas from academic papers for non-academics, from experiencing the problem to advisor-level review, focused on understanding over critique.
4. **Paper Tracing (ljg-paper-river)**: Reverse-read method that recursively mines precursor papers (up to 5 layers) + latest advances, narrating the evolution of a problem from its origins.
5. **Writing Engine (ljg-writes)**: Cut into an idea like a scalpel, peel layer by layer to the bottom. 1000-1500 word critical essays.
6. **Plain Language Engine (ljg-plain)**: Rewrite any content so a smart 12-year-old can understand it.
7. **Question Machine (ljg-qa)**: Extract core ideas into Q-A chains — questions that cut to the heart, answers in four parts (conclusion / formalization / steps / boundaries).
8. **Rank Reduction Engine (ljg-rank)**: Given a domain, find the irreducible independent generators behind it.
9. **Essence Arrow (ljg-think)**: Given an idea or phenomenon, drill vertically to the indivisible essence.
10. **Presentation Forger (ljg-present)**: Takahashi method (one keyword per slide) or Slogan style (color-block manifestos), outputting single-file HTML presentations.
11. **Reading Companion (ljg-read)**: Guide you through any text with three-layer English-to-Chinese translation (faithfulness-expressiveness-elegance) + structural annotation + deep questioning + cross-domain insights.
12. **Word Mastery (ljg-word)**: Deeply dissect a single English word's core semantics and epiphany moments.
13. **Investment Analysis (ljg-invest)**: Core judgment on whether a project is an "order-creating machine."
14. **Relationship Analysis (ljg-relationship)**: Five-layer structural diagnosis + psychoanalysis to help users "see" the real structure of their relationships.
15. **Roundtable Discussion (ljg-roundtable)**: Truth-oriented structured multi-person dialectical dialogue, generating ASCII thinking framework diagrams each round.
16. **Travel Research (ljg-travel)**: Input a city name, generate in-depth cultural research documents + portable cards.
17. **Skill Map (ljg-skill-map)**: Scan all installed skills and render a visual overview.
18. **Push Engine (ljg-push)**: One-click sync of local skills to a GitHub repository.
