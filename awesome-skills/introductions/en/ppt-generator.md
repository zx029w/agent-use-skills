# PPT Generator

**PPT Generator** is a skill that converts user scripts into Steve Jobs-style minimalist tech-themed HTML presentations with one click. It supports landscape fullscreen adaptive mode and outputs a single HTML file that can be run directly, giving your presentations a tech-forward and visually impactful feel.

## Tags

📊 Presentation | ✅ Verified

## Core Philosophy

- **Minimalism**: One idea per slide, avoiding information overload
- **Strong Visual Contrast**: Dark backgrounds with white text for visual impact
- **High Whitespace**: Rejecting dense layouts, letting content breathe
- **Strong Rhythm**: Carefully designed content pacing that keeps audiences engaged

## Key Features & Workflow

1. **Script Reading**: Intelligently reads user's original scripts or document content
2. **Content Refinement**: Simplifies content, enhances impact, adapts for presentation scenarios, outputs refined Markdown script
3. **Title Generation**: Generates titles for each section (≤12 characters), supporting contrast, question, assertion, number, and metaphor styles
4. **Structure Design**: Plans page sequence and types, referencing predefined slide types
5. **HTML Generation**: Generates complete HTML using templates, including animations and interactive effects
6. **Output Delivery**: Sequentially outputs refined script, slide structure outline, and complete HTML code
