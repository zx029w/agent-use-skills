# UI UX Pro Max

**UI UX Pro Max** is an AI skill that provides design intelligence for building professional UI/UX across multiple platforms and frameworks.

## Tags

🎨 Design & Creativity | ✅ Verified

## Core Philosophy

- **Design System Generation**: AI-powered reasoning engine that analyzes project requirements and generates complete, tailored design systems
- **Industry-Specific Rules**: 100 industry-specific reasoning rules covering SaaS, finance, healthcare, e-commerce, and more
- **Standardized Design Language**: Professional design resources including 67 UI styles, 96 color palettes, and 57 typography combinations

## Key Features & Workflow

1. **Design System Generator**: Input a product description (e.g., "Build a landing page for a beauty spa"), and the AI automatically recommends complete patterns, styles, colors, typography, and effects
2. **Intelligent Matching Engine**: Parallel search across product categories, style recommendations, color palettes, landing page patterns, and typography pairings to provide optimal design combinations
3. **Pre-Delivery Checklist**: Automatically validates against 99 UX guidelines and anti-pattern rules including contrast, responsiveness, and accessibility
4. **Multi-Stack Support**: Supports 13 tech stacks including React, Next.js, Vue, Nuxt.js, Svelte, Astro, SwiftUI, Jetpack Compose, React Native, and Flutter

## Skills Library Overview

- **67 UI Styles**: Including Minimalism, Neumorphism, Glassmorphism, Brutalism, Bento Grid, Dark Mode, AI-Native UI, and more
- **96 Color Palettes**: Industry-specific palettes for SaaS, E-commerce, Healthcare, Fintech, Beauty, and more
- **57 Font Pairings**: Curated Google Fonts typography combinations with import links
- **100 Reasoning Rules**: Design rules covering Tech, Finance, Healthcare, E-commerce, Services, Creative, and Emerging Tech industries
- **25 Chart Types**: Chart recommendations for dashboards and data analytics
