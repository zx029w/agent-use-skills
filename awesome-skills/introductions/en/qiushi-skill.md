# Qiushi Skill

**Qiushi Skill** is a collection of AI Agent Skills that distills one overarching principle and nine methodological tools from Mao Zedong's thought, systematically arming AI's mind. Not slogans, not motivational quotes, but operable methodology collections.

## Core Philosophy

- **Seek Truth from Facts**: Start from objectively existing actual things, let facts determine judgments, let reality correct theory
- **Perfectionism**: "The world fears nothing but the word 'serious'", completed solutions must be self-checked, never settle for "almost good enough"
- **Persistence**: "In difficult times, our comrades must see achievements, see light, and increase courage"

Each method is well-founded and traceable, directly quoting original texts from Mao's Selected Works (see `original-texts.md` in each skill directory).

## Key Features & Workflow

### Overarching Principle — Constraining All Judgment Processes

**☀️ Seek Truth from Facts**: Not the tenth ideological weapon, but the epistemological yardstick that all ideological weapons must obey. Start from objectively existing actual things, let facts determine judgments, let reality correct theory.

### Layer 1·Philosophical Foundation — Bottom Framework for Analyzing Any Problem

1. **⚔️ Contradiction Analysis**: Identify contradictions, grasp main contradictions, distinguish contradiction nature. "Once this main contradiction is grasped, all problems are readily solved."
2. **🔄 Practice-Cognition**: Practice → Cognition → Practice again, spiral upward. "Practice is the sole criterion for testing truth."

### Layer 2·Working Methods — Basic Methods for Daily Work

3. **🔎 Investigation**: No investigation, no right to speak. "Investigation is like 'ten months of pregnancy', solving problems is like 'one day of delivery'."
4. **👥 Mass Line**: From the masses, to the masses. Collect → Systematize → Return → Verify → Collect again.
5. **🪞 Criticism & Self-Criticism**: Punish past mistakes to prevent future ones, cure the sickness to save the person. "A house should be cleaned regularly."

### Layer 3·Strategic Tactics — Action Guidance for Specific Tasks

6. **⏳ Protracted Strategy**: Strategically despise, tactically value. Don't rush for quick success, don't fear difficulties.
7. **🎯 Concentrate Forces**: Better to cut off one finger than injure ten. Never fight unprepared battles.
8. **🔥 Spark Prairie Fire**: Establish base areas, don't be roaming insurgents. Start small, accumulate development.
9. **⚖️ Overall Planning**: Mobilize all positive factors. Reject one-sidedness, seek dynamic balance.

### Supporting File System

- **📜 Original Texts**: Each skill has `original-texts.md` with complete original citations
- **🤖 Subagent Prompts**: Dispatchable specialized agents (investigation-agent, contradiction-mapper, feedback-synthesizer)
- **🗺️ Reference Guides**: Contradiction types reference, work review checklist, protracted war phase assessment guide
