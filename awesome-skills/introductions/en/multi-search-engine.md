# Multi Search Engine

**Multi Search Engine** is a search engine integration tool requiring no API Keys, aggregating 8 domestic and 9 global search engines while supporting advanced search operators and privacy awareness.

## Core Philosophy
- **No API Key Dependency**: Performs direct web data fetching, lowering the barrier to entry.
- **Comprehensive Coverage**: Integrates mainstream engines across domestic and global spaces, meeting diverse regional and informational search needs.
- **Privacy First**: Natively supports privacy-conscious search engines like DuckDuckGo and Startpage to protect user tracking safely.
- **Advanced Retrieval Capabilities**: Supports complex search operators and time filters, enabling highly precise information targeting.

## Key Features & Workflow
1. **Multi-Engine Aggregation**: Provides ad-hoc access to up to 17 mainstream domestic and global search engines.
2. **Advanced Search Operators**: Seamlessly supports terms like `site:` (in-site query), `filetype:` (specific formats), `-` (exclude keyword), `""` (exact match), and `OR`.
3. **Time Filters**: Allows precise filtering of search results by specified time ranges (e.g., past hour `tbs=qdr:h`, day, week, month, year).
4. **DuckDuckGo Bangs Shortcuts**: Supports brief prefixes like `!g` or `!gh` for jumping seamlessly to designated platform eco-searches.
5. **WolframAlpha Knowledge Computation**: Enables direct answers for mathematical integrals, unit/currency conversions, or real-time queries for stocks and weather.

## Skills Library Overview
- **Domestic Search Engines**: Covers Baidu, Bing CN, 360, Sogou, WeChat, Toutiao, and Jisilu.
- **Global Search Engines**: Covers Google, Google HK, Bing INT, and Yahoo.
- **Privacy Search Engines**: Encompasses DuckDuckGo (no tracking), Startpage (Google results + privacy), Brave (independent index), Ecosia, and Qwant (EU GDPR compliant).
- **Knowledge Computation**: WolframAlpha (covering math, conversions, stocks, and weather).
