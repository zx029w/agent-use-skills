# Install Renwei Writing for OpenCode

## Prerequisites

- [OpenCode](https://opencode.ai) installed
- Git installed

## Installation Steps

### 1. Clone the renwei-writing repository

```bash
git clone https://github.com/orange2ai/renwei-writing.git ~/.config/opencode/skills/renwei-writing
```

### 2. Verify Installation

Restart OpenCode, then try asking:
- "Do you know Renwei Writing?"
- "Help me polish this paragraph"

If successful, OpenCode will automatically recognize and invoke the Renwei Writing skill.

## Updating

```bash
cd ~/.config/opencode/skills/renwei-writing
git pull
```

## Uninstallation

```bash
rm -rf ~/.config/opencode/skills/renwei-writing
```
