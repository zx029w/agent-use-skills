# 在 Cursor 中安装人味儿写作

## 前置条件

- [Cursor](https://cursor.com) 已安装
- Git 已安装

## 安装步骤

### 1. 克隆 renwei-writing 仓库

```bash
git clone https://github.com/orange2ai/renwei-writing.git ~/.cursor/skills/renwei-writing
```

### 2. 验证安装

重启 Cursor，然后尝试询问：
- “你会用人味儿写作吗？”
- “帮我润色这段文字”

如果安装成功，Cursor 会自动识别并调用人味儿写作技能。

## 更新

```bash
cd ~/.cursor/skills/renwei-writing
git pull
```

## 卸载

```bash
rm -rf ~/.cursor/skills/renwei-writing
```
