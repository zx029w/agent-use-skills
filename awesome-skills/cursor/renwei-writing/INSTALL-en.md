# Install Renwei Writing for Cursor

## Prerequisites

- [Cursor](https://cursor.com) installed
- Git installed

## Installation Steps

### 1. Clone the renwei-writing repository

```bash
git clone https://github.com/orange2ai/renwei-writing.git ~/.cursor/skills/renwei-writing
```

### 2. Verify Installation

Restart Cursor, then try asking:
- "Do you know Renwei Writing?"
- "Help me polish this paragraph"

If successful, Cursor will automatically recognize and invoke the Renwei Writing skill.

## Updating

```bash
cd ~/.cursor/skills/renwei-writing
git pull
```

## Uninstallation

```bash
rm -rf ~/.cursor/skills/renwei-writing
```
