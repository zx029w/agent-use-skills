# Install Renwei Writing for Claude Code

## Prerequisites

- [Claude Code](https://claude.ai/code) installed
- Git installed

## Installation Steps

### 1. Clone the renwei-writing repository

```bash
git clone https://github.com/orange2ai/renwei-writing.git ~/.claude/skills/renwei-writing
```

### 2. Verify Installation

Restart Claude Code, then try asking:
- "Do you know Renwei Writing?"
- "Help me polish this paragraph"

If successful, Claude Code will automatically recognize and invoke the Renwei Writing skill.

## Updating

```bash
cd ~/.claude/skills/renwei-writing
git pull
```

## Uninstallation

```bash
rm -rf ~/.claude/skills/renwei-writing
```
