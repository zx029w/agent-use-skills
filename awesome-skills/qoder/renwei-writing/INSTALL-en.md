# Install Renwei Writing for Qoder

## Prerequisites

- [Qoder](https://qoder.ai) installed
- Git installed

## Installation Steps

### 1. Clone the renwei-writing repository

```bash
git clone https://github.com/orange2ai/renwei-writing.git ~/.qoder/skills/renwei-writing
```

### 2. Verify Installation

Restart Qoder, then try asking:
- "Do you know Renwei Writing?"
- "Help me polish this paragraph"

If successful, Qoder will automatically recognize and invoke the Renwei Writing skill.

## Updating

```bash
cd ~/.qoder/skills/renwei-writing
git pull
```

## Uninstallation

```bash
rm -rf ~/.qoder/skills/renwei-writing
```
