# Install Renwei Writing for OpenAgent

## Prerequisites

- [OpenAgent](https://openagent.ai) installed
- Git installed

## Installation Steps

### 1. Clone the renwei-writing repository

```bash
git clone https://github.com/orange2ai/renwei-writing.git ~/.openagent/skills/renwei-writing
```

### 2. Verify Installation

Restart OpenAgent, then try asking:
- "Do you know Renwei Writing?"
- "Help me polish this paragraph"

If successful, OpenAgent will automatically recognize and invoke the Renwei Writing skill.

## Updating

```bash
cd ~/.openagent/skills/renwei-writing
git pull
```

## Uninstallation

```bash
rm -rf ~/.openagent/skills/renwei-writing
```
