# 装修避坑技能集

本技能集包含 11 个面向中文装修场景的 AI Agent 技能，知识来源为「从学到装｜装修课堂」20 年实战经验。

## 包含技能

- 全屋定制不踩雷（`quanwu-dingzhi-bucailei`）
- 老房改造不踩雷（`laofang-gaizao-bucailei`）
- 装修合同审核（`zhuangxiu-hetong-shenhe`）
- 装修报价审核（`zhuangxiu-baojia-shenhe`）
- 装修方案诊断（`zhuangxiu-fangan-zhenduan`）
- 装修环保避坑（`zhuangxiu-huanbao-bikeng`）
- 装修流程不踩坑（`zhuangxiu-liucheng-bucai`）
- 装修翻车急救（`zhuangxiu-fanche-jijiu`）
- 装修避坑问答（`zhuangxiu-bikeng-wenda`）
- 装修增项避雷（`zhuangxiu-zengxiang-bilei`）
- 装修公司避雷指南（`zhuangxiu-gongsi-bilei`）

## 安装方式

```bash
git clone https://github.com/zx029w/zhuangxiu-skills.git ~/.zhuangxiu-skills
```

将各技能子目录链接到对应 Agent 框架的 skills 目录即可。详见各平台安装指南。
