# Install Renwei Writing for OpenClaw

## Prerequisites

- [OpenClaw](https://openclaw.ai) installed
- Git installed

## Installation Steps

### 1. Clone the renwei-writing repository

```bash
git clone https://github.com/orange2ai/renwei-writing.git ~/.openclaw/skills/renwei-writing
```

### 2. Verify Installation

Restart OpenClaw, then try asking:
- "Do you know Renwei Writing?"
- "Help me polish this paragraph"

If successful, OpenClaw will automatically recognize and invoke the Renwei Writing skill.

## Updating

```bash
cd ~/.openclaw/skills/renwei-writing
git pull
```

## Uninstallation

```bash
rm -rf ~/.openclaw/skills/renwei-writing
```
